# インターン体験記と会社ページとの乖離度測定プログラム

会社が採用ページで謳う「社風・文化」と、インターン生が実際に体験したこととのギャップを自動測定するWebアプリです。

---

## 起動手順

```bash
# 1. 依存パッケージをインストール
pip install -r requirements.txt

# 2. サンプルデータを生成（20社分）
python collect.py

# 3. CLI でランキングを確認（動作確認）
python engine.py

# 4. Web UI を起動
python -m streamlit run app.py
```

ブラウザが自動で開き `http://localhost:8501` にアクセスできます。

> ⚠️ `streamlit run app.py` ではなく `python -m streamlit run app.py` を使ってください

---

## 測定する4つの指標

| 指標 | 計算方法 | 意味 |
|------|----------|------|
| **乖離度スコア** | 1 − コサイン類似度（TF-IDF） | 高いほど建前と本音がズレている |
| **感情スコア** | pn-ja感情極性辞書 + 内蔵辞書 | 正=ポジティブ体験 / 負=ネガティブ体験 |
| **KW一致率** | 特徴語の重複率 | 低いほど語彙レベルで乖離している |
| **社風リアル度** | 上記3指標の重み付き統合 | 高いほど「会社説明が実態に近い」 |

```
社風リアル度 = (1 − 乖離度) × 0.5 + 感情正規化 × 0.3 + KW一致率 × 0.2
```

## ファイル構成

```
skill_matcher/
├── collect.py        # データ収集（robots.txt確認付きスクレイピング + サンプル20社）
├── engine.py         # 乖離度計算コア（形態素解析 / TF-IDF / 感情 / KW差分）
├── app.py            # Streamlit UI（全社ランキング / 企業詳細 / 手動入力）
├── jobs.csv          # 収集した会社ページ & 体験記データ
├── pn-ja.dic         # 感情極性辞書（任意。なくても内蔵辞書で動作）
└── requirements.txt  # 依存パッケージ
```

## UIの3モード

### 1. 全社ランキング
- 社風リアル度の低い順（＝要注意企業）にランキング表示
- 各社の乖離度・感情スコア・KW一致率を一覧で比較
- 会社ページ固有キーワード（建前ワード）と体験記固有キーワード（本音ワード）を表示

### 2. 企業詳細分析
- 特定企業の詳細スコアと代表文を表示
- 会社ページ ↔ 体験記のキーワード差分を視覚化

### 3. 手動入力で分析
- 採用ページと体験記のテキストをコピペするだけで即座に乖離度を測定
- ワンキャリア・就活会議などから体験記を貼り付けて使用可能

## ⚙️ 形態素解析（MeCab対応）

| 環境 | 動作 |
|------|------|
| MeCab インストール済み | 名詞・動詞・形容詞を正確に抽出 |
| fugashi インストール済み | fugashi で形態素解析 |
| **どちらもなし（デフォルト）** | **char n-gram（2〜3文字）で代替。追加インストール不要** |

```bash
# 精度向上したい場合（任意）
pip install fugashi unidic-lite
```
